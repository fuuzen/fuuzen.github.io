# 关于本站

这里是我的个人主页，放一些技术、学习、ACGN 等乱七八糟东西的分享。原本这里收录了一些学习笔记、思维导图、资料甚至实验报告，但目前已经与本博客解耦，放在了我的笔记本里，目前和本博客均基于 (白嫖) Github Page 部署，这个笔记本相当于本站的 /notebook 二级分站：

風船の笔记本

笔记有问题欢迎联系我喵

要问为什么我不完整地自己租云服务器自建一个博客呢？因为我没钱并且懒因为 github.io 多好用啊，带宽又足，CDN 加速也给力，反正就是个静态网页就这样吧，目前来看很多大项目的文档甚至官网都基于 Github Page 和 github.io ，我大概还能白嫖很长一段时间。至于个性化域名，我买了个 taf.fyi 域名，比较喜欢短的域名，而且似乎单纯作为 github.io 子域名被搜索引擎收录比较困难，所以本站的域名就干脆用这个了。

本站在一个疑似已停止维护的 Hexo 主题 Shoka 基础上魔改而来，并自己维护。
