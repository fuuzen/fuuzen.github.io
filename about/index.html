要问为什么我不完整地自己租云服务器自建一个博客呢？因为我没钱并且懒因为 github.io 多好用啊，带宽又足，CDN 加速也给力，反正就是个静态网页就这样吧，目前来看很多大项目的文档甚至官网都基于 Github Page 和 github.io ，我大概还能白嫖很长一段时间。至于个性化域名，我买了个 taf.fyi 域名一年来着，比较喜欢短的域名，而且似乎单纯作为 github.io 子域名被搜索引擎收录比较困难，所以本站的域名就干脆用这个了。

本站在一个疑似已停止维护的 Hexo 主题 Shoka 基础上魔改而来，并自己维护。其实 Shoka 有一个正在维护的精神续作 ShokaX 主题，但我更想自己搞，而且我觉得 Shoka「書架(しょか)」比 ShokaX 好听。
