# зј–з Ғзҡ„жҰӮеҝө

е№ҝд№үзҡ„зј–з ҒжҢҮзҡ„е°ұжҳҜе°ҶдҝЎжҒҜд»ҺдёҖз§ҚеҪўејҸиҪ¬жҚўдёәеҸҰдёҖз§Қи§„иҢғзҡ„еҪўејҸпјӣ

еҜ№дәҺж•°еӯ—з”өи·ҜпјҢе°ұжҳҜе°ҶзҺ°е®һдё–з•Ңдёӯзҡ„еҗ„з§Қзү©зҗҶдҝЎжҒҜиҪ¬жҚўдёәзҰ»ж•Јзҡ„з”өеҺӢеәҸеҲ—пјҢжқҘиЎЁзӨәгҖҒеӨ„зҗҶе’ҢеҲҶжһҗдҝЎжҒҜпјӣ

дәӢе®һдёҠпјҢиҝӣдҪҚи®Ўж•°еҲ¶д№ҹжҳҜдёҖз§Қзј–з ҒпјҢз”ЁдёҚеҗҢиҝӣеҲ¶ж•°жқҘиЎЁзӨә вҖңйҮҸвҖқпјӣ

е°Ҷж•°еҲ¶дёҺз ҒеҲ¶еҲҶејҖд»Ӣз»ҚпјҢйҰ–е…ҲеҚ•зӢ¬и®Іи§Јж•°еҲ¶жҳҜеӣ дёәж•°еҲ¶зҡ„дёҖдёӘйҮҚиҰҒзҡ„зү№зӮ№жҳҜе…¶зј–з ҒеҗҺиғҪеӨҹеҸӮдёҺз®—жңҜиҝҗз®—пјӣиҖҢдёҖиҲ¬жқҘиҜҙз ҒеҲ¶жҳҜеҜ№йқһж•°еҖјдҝЎжҒҜзҡ„иЎЁзӨәпјӣ

ж•ҷжқҗдёӯзҡ„зј©еҶҷжңҜиҜӯпјҡ

  • MSB: Most Significant BitпјҢжңҖй«ҳдҪҚ
  • LSB: Least Significant BitпјҢжңҖдҪҺдҪҚ

# дәҢиҝӣеҲ¶зј–з Ғ

ж— з¬ҰеҸ·дәҢиҝӣеҲ¶зј–з ҒеҸӘиғҪиЎЁзӨә 0 е’ҢжӯЈж•°пјҢиҝҷйҮҢжҺҘдёӢжқҘи®Ёи®әзҡ„йғҪжҳҜжңүз¬ҰеҸ·ж•°гҖӮ

пјҲжңүз¬ҰеҸ·пјүдәҢиҝӣеҲ¶ж•°пјҡеңЁеүҚйқўеўһеҠ дёҖдёӘз¬ҰеҸ·дҪҚпјҢ0 иЎЁзӨәжӯЈж•°пјҢ1 иЎЁзӨәиҙҹж•°пјӣеј•е…Ҙз¬ҰеҸ·дҪҚеҗҺпјҢдёҖдёӘдәҢиҝӣеҲ¶ж•°зҡ„жңҖй«ҳдҪҚжҳҜз¬ҰеҸ·дҪҚпјҢз¬ҰеҸ·дҪҚзҡ„ 0 / 1 дёҚд»ЈиЎЁеӨ§е°ҸпјҢеҸӘжҳҜд»ЈиЎЁжӯЈиҙҹ

жңүз¬ҰеҸ·дәҢиҝӣеҲ¶ж•°зҡ„дёүз§Қзј–з ҒиЎЁзӨәпјҡ

  • еҺҹз Ғ вҖ”вҖ”Sign Magnitude Representation

    • жӯЈж•°е’Ң 0пјҡжҷ®йҖҡзҡ„дәҢиҝӣеҲ¶ж•°иЎЁзӨә
    • иҙҹж•°пјҡз»қеҜ№еҖјзҡ„дәҢиҝӣеҲ¶ж•°иЎЁзӨәеҠ дёҠиҙҹеҸ·

  • еҸҚз Ғ вҖ”вҖ”One's Complement

    • жӯЈж•°е’Ң 0пјҡ = еҺҹз Ғ
    • иҙҹж•°пјҡеҺҹз ҒжүҖжңүж•°еӯ—дҪҚеҸ–еҸҚ

  • иЎҘз Ғ вҖ”вҖ”Two's Complement

    • жӯЈж•°е’Ң 0пјҡ = еҺҹз Ғ
    • иҙҹж•°пјҡ= еҸҚз Ғ + 1

иЎҘз Ғзҡ„иЎҘз ҒеҚідёәеҺҹз Ғ

е…ідәҺеҺҹз ҒгҖҒеҸҚз ҒгҖҒиЎҘз Ғзҡ„жҳ“й”ҷзӮ№пјҡ

  • еҸҚз ҒеҸ–еҸҚжҳҜжҢҮеҜ№ж•°еӯ—дҪҚеҸ–еҸҚпјҢдёҚж”№еҸҳз¬ҰеҸ·дҪҚпјҒ
    • дҫӢеҰӮ -7 еҚі 1111 зҡ„еҸҚз ҒжҳҜ 1000пјҢдёҚжҳҜ 0000 пјҒ
  • иЎҘз ҒжҳҜдёҚеҜ№з§°зҡ„пјҢдҫӢеҰӮиЎҘз Ғдёә 1000 д»ЈиЎЁ - 8 пјҢиҖҢдёҚд»ЈиЎЁ - 0 пјҒ еҚіеҗҢж ·дҪҚж•°зҡ„дәҢиҝӣеҲ¶иЎҘз ҒпјҢиҙҹж•°дјҡжҜ”жӯЈж•° вҖңеӨҡдёҖдёӘвҖқпјӣ
  • иЎҘз Ғзҡ„ вҖңеҠ  1вҖқ жҢҮзҡ„жҳҜеңЁжңҖдҪҺдҪҚеҠ  1 зҡ„дәҢиҝӣеҲ¶еҠ жі•иҝҗз®—пјҢдёҚжҳҜжҢҮж•°еҖј 1пјҢеҚіеҰӮжһңеҗ«жңүе°Ҹж•°йғЁеҲҶпјҢ1 жҳҜеҠ еңЁж•ҙдҪ“зҡ„жңҖеҸіиҫ№пјӣ

  1. еҲҶжһҗйңҖиҰҒдҪҝз”ЁеӨҡе°‘дёӘж•°еӯ—дҪҚпјӣпјҲеҹ№е…»жҖқз»ҙд№ жғҜпјҒдёҚиғҪеҝҪз•ҘпјҒпјү
  2. з”ЁеёҰз¬ҰеҸ·дҪҚзҡ„дәҢиҝӣеҲ¶ж•°иЎЁзӨәпјҢжұӮи§ЈжҜҸдёӘдәҢиҝӣеҲ¶ж•°зҡ„иЎҘз Ғпјӣ
  3. еҜ№иЎҘз ҒиҝӣиЎҢеҠ жі•иҝҗз®—пјҢеҶҚеҜ№з»“жһңжұӮиЎҘз ҒпјҢеҚідёәжңҖз»Ҳз»“жһңпјӣ

жіЁж„ҸжңҖеҗҺдёҖжӯҘпјҢеҜ№иЎҘз ҒиҝӣиЎҢеҠ жі•иҝҗз®—ж—¶пјҢж•°еӯ—дҪҚжңҖй«ҳдҪҚдә§з”ҹзҡ„иҝӣдҪҚиҰҒиҝӣеҲ°з¬ҰеҸ·дҪҚдёҠеҺ»пјҒпјҲж•ҷжқҗдёӯжҸҸиҝ°зҡ„ вҖңиҲҚејғдә§з”ҹзҡ„иҝӣдҪҚвҖқ жҢҮзҡ„жҳҜиҲҚејғз¬ҰеҸ·дҪҚдә§з”ҹзҡ„иҝӣдҪҚпјү

# иЎҘз Ғзҡ„жү©еұ•

е°ҶдёҖдёӘ n дҪҚиЎҘз Ғжү©еұ•еҲ° n+k дҪҚпјҢеҸӘйңҖе°ҶжңҖй«ҳдҪҚз¬ҰеҸ·дҪҚеӨҚеҲ¶ k дёӘеүҚзҪ®гҖӮ

# еҸҚз Ғзҡ„еҠ жі•

иӢҘжңҖй«ҳдҪҚдә§з”ҹиҝӣдҪҚпјҢйңҖиҰҒеҫӘзҺҜиҝӣдҪҚпјҢеҚіжңҖдҪҺдҪҚ вҖңиҝӣдёҖдҪҚвҖқпјҲ +1 пјүгҖӮ

# ж–№жі•дәҢпјҡ

д»ҺеҸіеҗ‘е·ҰжүҫеҲ°з¬¬дёҖдёӘ 1 пјҢиҝҷдёӘ 1 еүҚйқўзҡ„жүҖжңүдҪҚеҸ–еҸҚ

иӢҘжІЎжңү 1 пјҢд№ҹе°ұжҳҜ 0 пјҢеҺҹз Ғ = иЎҘз Ғ

# иҙҹж•°пјҡиЎҘз Ғ -> еҺҹз Ғ

иҝҷйҮҢеҸӘи®Ёи®әиҙҹж•°зҡ„иҪ¬жҚўпјҢжӯЈж•°дёҚйңҖиҰҒиҪ¬жҚў

# еҚҒиҝӣеҲ¶ж•°зҡ„дәҢиҝӣеҲ¶д»Јз Ғ

еҚҒиҝӣеҲ¶д»Јз Ғз”ЁеӣӣдҪҚдәҢиҝӣеҲ¶ж•°пјҲжңҖеӨҡеҸҜиЎЁзӨә 24=162^4=1624=16 дёӘпјүжқҘиЎЁзӨә 0 ~ 9 пјҢжІЎжңү 10 вҡ пёҸ

# BCD/8421

8421 з Ғд№ҹеҸ« BCD з Ғ (Binary Coded Decimal)гҖӮеҸҜд»Ҙи§ҶдёәдёҖз§ҚеҺӢзј©зј–з ҒпјҲзӣёжҜ” int е’Ң char пјүгҖӮ

дҪҷ 3 з ҒжҳҜеңЁ BCD з ҒеҚі 8421 з Ғзҡ„еҹәзЎҖдёҠ +3 пјҢеҚіз”Ё 3 ~ 12 еҜ№еә”зҡ„дәҢиҝӣеҲ¶ж•°жқҘиЎЁзӨә 0 ~ 9 пјӣ

дҪҷ 3 еҫӘзҺҜз ҒжҳҜеҸ–ж јйӣ·з Ғдёӯзҡ„ 3 ~ 12 пјӣ

# ж јйӣ·з Ғ

ж•ҷжқҗдёӯзҡ„ж јйӣ·з Ғзү№жҢҮзҡ„жҳҜеӣӣдҪҚж јйӣ·з ҒпјҢеҚіз”ЁеӣӣдҪҚд»Јз ҒжқҘиЎЁзӨә 0 пҪһ 15 е…ұеҚҒе…ӯз§ҚзҠ¶жҖҒпјӣ

# еӣӣз»ҙж јйӣ·з ҒиЎЁ

йңҖиҰҒзҶҹз»ғжҺҢжҸЎпјҒйҮҚзӮ№жҳҜзҶҹжӮүзј–з Ғзҡ„и§„еҲҷе’Ңзү№зӮ№

# дәҢиҝӣеҲ¶з ҒдёҺж јйӣ·з Ғзҡ„жҳ е°„

  • жңҖй«ҳдҪҚзӣёеҗҢ

ж јйӣ·з ҒиҪ¬жҚўжҲҗдәҢиҝӣеҲ¶з Ғпјҡ

  • жңҖй«ҳдҪҚзӣёеҗҢ
  • ж јйӣ·з Ғд»Һе·ҰеҲ°еҸіпјҢе°ҶдәҢиҝӣеҲ¶з ҒдёҠдёҖдҪҚ A пјҢж јйӣ·з ҒеҪ“еүҚдҪҚ B дёӨдҪҚејӮжҲ–пјҢеҫ—еҲ°дәҢиҝӣеҲ¶з ҒеҜ№еә”зҡ„ B дҪҚ
