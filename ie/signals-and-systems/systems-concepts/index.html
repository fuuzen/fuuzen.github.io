# 对于连续系统

步骤如下

  1. 首先判断其阶数，阶数 = 积分器个数
  2. 设信号传递最后一个积分器输出为 x(t) ，则各积分器的输入输出均可确定
  3. 对每个加法器列⽅程：输入和 = 输出和
  4. 联立方程消去中间变量 x(t) ，得到关于输出信号 y(t) 的微分方程
