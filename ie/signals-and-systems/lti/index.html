Linear Time Invariant 线性时不变

基本信号应具有以下特点：

  • 本身尽可能简单，并且用它们的线性组合能够表示尽可能广泛的信号
  • LTI 系统对这种信号的响应易于计算且有利于对系统的深入理解

两类有用的基本信号：

  • 复指数信号
  • 单位冲激信号

本章主要考虑以单位冲激信号为基本信号进行信号与系统分析

# 卷积和

# 定义

对于任意一个离散时间系统：

设离散时间系统为 LTI 的，定义单位脉冲响应 h[n]h[n]h[n] 为以单位冲激信号 δ[n]\delta[n]δ[n] 为输入时的输出：

δ[n]→h[n]\delta[n]\to h[n]δ[n]→h[n]

由 LTI 时不变性有：

δ[n−k]→h[n−k]\delta[n-k]\to h[n-k]δ[n−k]→h[n−k]

由 LTI 线性有：

x[n]=∑k=−∞+∞x[k]δ[n−k]→y[n]=∑k=−∞+∞x[k]h[n−k]x[n]=\sum_{k=-\infin}^{+\infin}x[k]\delta[n-k]\to y[n]=\sum_{k=-\infin}^{+\infin}x[k]h[n-k]x[n]=k=−∞∑+∞​x[k]δ[n−k]→y[n]=k=−∞∑+∞​x[k]h[n−k]

这种求得系统响应的运算关系称为卷积和，即离散时间 LTI 系统的响应可以用卷积和表示为:

x[n]=x[n]∗h[n]=∑k=−∞+∞x[k]h[n−k]x[n]=x[n]*h[n]=\sum_{k=-\infin}^{+\infin}x[k]h[n-k]x[n]=x[n]∗h[n]=k=−∞∑+∞​x[k]h[n−k]

更一般的形式：?

x[n−n0−n1]=x[n−n0]∗h[n−n1]x[n-n_0-n_1]=x[n-n_0]*h[n-n_1]x[n−n0​−n1​]=x[n−n0​]∗h[n−n1​]

⭐️只要得到了 LTI 系统对 δ[n]\delta[n]δ[n] 的 响 应 h[n]h[n]h[n] —— 单位脉冲响应，就可以得到 LTI 系统对任何输入信号 x[n]x[n]x[n] 的响应。

换言之，一个 LTI 系统可以完全由它的单位脉冲响应 h[n]h[n]h[n] 来表征。

# 特殊离散 LTI 系统

# 延时器

h[n]=δ[n−n0]h[n]=\delta[n-n_0]h[n]=δ[n−n0​]

具有该单位脉冲响应的系统一般情况下存在多种可能性，但是对应的 LTI 系统只有一个：

y[n]=x[n−n0]=x[n]∗δ[n−n0]y[n]=x[n-n_0]=x[n]*\delta[n-n_0]y[n]=x[n−n0​]=x[n]∗δ[n−n0​]

# 累加器

h[n]=∑k=−∞nδ[k]=u[n]h[n]=\sum_{k=-\infin}^{n}\delta[k]=u[n]h[n]=k=−∞∑n​δ[k]=u[n]

对应的 LTI 系统：

y[n]=x[n]∗u[n]=∑k=−∞nx[k]y[n]=x[n]*u[n]=\sum_{k=-\infin}^{n}x[k]y[n]=x[n]∗u[n]=k=−∞∑n​x[k]

# 计算

选择 nnn 值并固定，求和项视为 kkk 的函数，步骤如下：

  1. 将 x[n]x[n]x[n] 与 h[n]h[n]h[n] 变为 x[k]x[k]x[k] 与 h[k]h[k]h[k] ，以 kkk 为求和变量
  2. 将 h[k]h[k]h[k] 翻转变为 h[−k]h[-k]h[−k]；
  3. 将 h[−k]h[-k]h[−k] 平移 nnn，变为 h[n−k]h[n - k]h[n−k]；
  4. 将 x[k]x[k]x[k] 和 h[n−k]h[n-k]h[n−k] 相乘；
  5. 对乘积 x[k]h[n−k]x[k]h[n-k]x[k]h[n−k] 在 k∈Zk\in \mathbb{Z}k∈Z 取和；
    • 一般都转化为只对非 0 区间积分。注意非 0 区间上下限！！！
  6. 改变 nnn 值并重复 3–5 步，直到 nnn 超过了 x[n]x[n]x[n] 非零部分定义域

注意很多题目最后答案会加上一个 u(n)u(n)u(n) ，是因为只求定义域在 [0,+∞)[0,+\infin)[0,+∞) 的部分

# 卷积积分

# 定义

类似积分思想，转化为无限多的离散和的极限

用冲激信号表示连续时间信号：

将输入信号 x(t)x(t)x(t) 近似表示为移位、放缩后的脉冲之和

当 Δ → 0 时 :

  • xΔ(t)→x(t)x_Δ(t) → x(t)xΔ​(t)→x(t)
  • yΔ(t)→y(t)y_Δ(t) → y(t)yΔ​(t)→y(t)

x(t)=∫−∞+∞x(τ)δ(t−τ)dτ=x(t)∗δ(t)y(t)=∫−∞+∞x(τ)h(t−τ)dτ=x(t)∗h(t)x(t)=\int_{-\infin}^{+\infin}x(\tau)\delta(t-\tau)d\tau=x(t)*\delta(t)\\ y(t)=\int_{-\infin}^{+\infin}x(\tau)h(t-\tau)d\tau=x(t)*h(t)x(t)=∫−∞+∞​x(τ)δ(t−τ)dτ=x(t)∗δ(t)y(t)=∫−∞+∞​x(τ)h(t−τ)dτ=x(t)∗h(t)

  • 其中第一个式子意义是：用位移的单位冲激信号表示任意的连续时间信号

求卷积积分 x(t)∗h(t)=∫−∞+∞x(τ)h(t−τ)dτx(t) * h(t) = \int_{-∞}^{+∞}x(\tau)h(t - \tau)d\taux(t)∗h(t)=∫−∞+∞​x(τ)h(t−τ)dτ 的过程与卷积和计算过程相似。卷积积分的计算步骤如下：

# 特殊连续 LTI 系统

# 积分器

单位冲激信号为输入下系统的响应为：

h(t)=u(t)h(t)=u(t)h(t)=u(t)

对任意激励信号 x(t)x(t)x(t) ，系统响应为：

y(t)=x(t)∗h(t)=∫−∞+∞x(τ)h(t−τ)dτ=∫−∞tx(τ)dτy(t)=x(t)*h(t)=\int_{-\infin}^{+\infin}x(\tau)h(t-\tau)d\tau = \int_{-\infin}^{t}x(\tau)d\tauy(t)=x(t)∗h(t)=∫−∞+∞​x(τ)h(t−τ)dτ=∫−∞t​x(τ)dτ

# 级联可交换

级联次序可交换的前提条件:

  • 系统是 LTI 的
  • 所有涉及到的卷积运算必须收敛

# LTI 系统的性质

# 记忆性

对于离散时间 LTI 系统：

y[n]=∑k=−∞+∞x[k]h[n−k]y[n]=\sum_{k=-\infin}^{+\infin}x[k]h[n-k]y[n]=k=−∞∑+∞​x[k]h[n−k]

如果系统是无记忆的，则在任何时刻 n ，y [n] 都只能和 n 时刻的输入有关，等式右边的和式中只能有 k = n 时的一项非零，即必须有 h [n - k] = 0 , k ≠ n，也即：

h[n]=0,n≠0h[n] = 0 ,\ \ \ n\neq0h[n]=0,  n=0

所以，离散时间无记忆系统的单位脉冲响应为：

h[n]=kδ[n]h[n]=k\delta[n]h[n]=kδ[n]

即为离散时间系统无记忆的充要条件。

类似地，连续时间系统无记忆的充要条件为：

h(n)=kδ[(n)h(n)=k\delta[(n)h(n)=kδ[(n)

当 k = 1 时系统是恒等系统。

# 可逆性

如果 LTI 系统是可逆的，则其逆系统也是 LTI 系统。

互逆的两个系统的满足（充要条件）卷积和等于单位冲激信号：

h(t)∗g(t)=δ(t)h[n]∗g[n]=δ[n]\begin{aligned} h(t) * g(t) =\delta(t)\\ h[n] * g[n] =\delta[n] \end{aligned}h(t)∗g(t)=δ(t)h[n]∗g[n]=δ[n]​

从而有：

x(t)∗h(t)∗g(t)=x(t)∗δ(t)=x(t)x[t]∗h[n]∗g[n]=x[t]∗δ[n]=x[t]\begin{aligned} x(t)*h(t)*g(t)=x(t)*\delta(t)=x(t)\\ x[t]*h[n]*g[n]=x[t]*\delta[n]=x[t] \end{aligned}x(t)∗h(t)∗g(t)=x(t)∗δ(t)=x(t)x[t]∗h[n]∗g[n]=x[t]∗δ[n]=x[t]​

例如：延时器是可逆的 LTI 系统，h (t) = \delta (t - t0)，其逆系统是 g (t) = \delta (t + t0)，显然有：

h(t)∗g(t)=(t−t0)∗δ(t+t0)=δ(t)h(t) * g(t) = (t - t_0) * \delta(t + t_0) = \delta(t)h(t)∗g(t)=(t−t0​)∗δ(t+t0​)=δ(t)

又如：累加器是可逆的 LTI 系统，其 h [n] = u [n]，其逆系统是 g [n] = \delta [n] - \delta [n - 1]，显然也有：

h[n]∗g[n]=u[n]∗(δ[n]−δ[n−1])=u[n]−u[n−1]=δ[n]\begin{aligned} h[n] * g[n] &= u[n] * (\delta[n] - \delta[n - 1])\\ &= u[n] - u[n - 1] = \delta[n] \end{aligned}h[n]∗g[n]​=u[n]∗(δ[n]−δ[n−1])=u[n]−u[n−1]=δ[n]​

# 因果性

对于离散时间 LTI 系统：

y[n]=∑k=−∞+∞x[k]h[n−k]y[n]=\sum_{k=-\infin}^{+\infin}x[k]h[n-k]y[n]=k=−∞∑+∞​x[k]h[n−k]

因果性意味着上式等号右边和式中所有 k > n 的项都必须为零，即 h [n - k] = 0，k > n，也即：

h[n]=0,n<0h[n] = 0,\ \ \ n < 0h[n]=0,  n<0

同理对连续时间因果系统有：

h(t)=0,t<0h(t) = 0, \ \ \ t < 0h(t)=0,  t<0

即为 LTI 系统具有因果性的充要条件

对于线性系统而言，因果性与初始松弛条件等价。所谓系统满足初始松弛条件是指该系统具有如下表现：对于任意时刻，只要该时刻之前系统的输入为 0，则该时刻之前系统的输出也必然为 0。

– 该等价性的前提是系统为线性的。否则不一定成立。例如 y n = 2x n + 3 是因果的，但显然不满足初始松弛条件。

# 线性常系数微分方程

一般情况 N≥MN\geq MN≥M , N 阶非其次常系数微分方程的一般形式：

∑k=0Nakdky(t)dtk=∑k=0Mbkdky(t)dtk\sum_{k=0}^{N}a_k\frac{d^ky(t)}{dt^k}=\sum_{k=0}^{M}b_k\frac{d^ky(t)}{dt^k}k=0∑N​ak​dtkdky(t)​=k=0∑M​bk​dtkdky(t)​

其中 ak, bk 均为常数

求解该微分方程（求 y (t) 的显式表达）的步骤：

  1. 求出齐次解（通解）yh (t) ，也称为系统的自然响应、自由响应
    1. 建立特征方程求出特征根 𝜆k
    2. 根据特征根得到齐次解形式
    3. 根据附加条件确定齐次解形式中的待定常数 Ck
  2. 求特解 y𝑝(t)，也称系统的受迫响应、强迫响应，其形式与输入信号有关
    1. 根据输入信号形式得到特解形式
    2. 将输入信号与特解代入微分方程确定特解形式中的待定常数
  3. 得到完全解 y (t) = yh(t) + y𝑝(t)

对于线性系统，初始松弛条件与满足因果性是等价的
