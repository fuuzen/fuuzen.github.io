# 关于挑战实验

阅读了 egos-2000 操作系统的文档和仓库，按照其教程实现了操作系统映像文件的编译和写入 microSD 卡的过程

但是 egos-2000 操作系统暂时只支持 Arty A7-35T board、A7-100T board 和 S7-50 board，移植到 z710 上比较困难，所以没能成功
