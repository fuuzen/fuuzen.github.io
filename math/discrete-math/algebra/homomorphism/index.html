# 群的同态

# 定义

设 GGG 与 G′G'G′ 是两个群，函数 φ:G→G′φ:G→G'φ:G→G′ 如果满足：

∀a,b∈G,φ(ab)=φ(a)φ(b)∀a,b\in G,\ \ φ(ab)= φ(a)φ(b)∀a,b∈G, φ(ab)=φ(a)φ(b)

则 φφφ 是群 GGG 到 G′G'G′ 的同态 (homomorphism)（一个函数）

φ(a)φ(b)φ(a)φ(b)φ(a)φ(b) 是 G′G'G′ 中的运算，而 ababab 是 GGG 中的运算

考察同态 φ\varphiφ 作为函数的性质，可以将同态分为：

  • 如果 G=G′G=G'G=G′ ，则称 φφφ 为自同态 (automorphism)；
  • 如果同态 φ:G→G′φ:G→G'φ:G→G′ 是满函数，则称为满同态 (epimorphism)，并称 GGG 与 G′G'G′ 同态，记为φ:G∼G′φ:G∼G'φ:G∼G′；
  • 如果同态 φ:G→G′φ:G→G'φ:G→G′ 是单函数，则称为单同态 (monomorphism)；
  • 如果同态 φ:G→G′φ:G→G'φ:G→G′ 是双函数，则称为同构 (isomorphism)，并称群 GGG 与 G′G'G′ 同构，记为 φ:G≅G′φ:G≅G'φ:G≅G′；

# 群同态的单位元

若 eee 是 GGG 的单位元，则 φ(e)\varphi(e)φ(e) 是 G′G'G′ 的单位元；

设 G′G'G′ 的单位元 e′e'e′ ，则:

φ(e)φ(e)=φ(ee)=φ(e)=φ(e)e′\varphi(e)\varphi(e)=\varphi(ee)=\varphi(e)=\varphi(e)e'φ(e)φ(e)=φ(ee)=φ(e)=φ(e)e′

从而由群的运算满足消去律得 φ(e)=e′\varphi(e)=e'φ(e)=e′ 。
