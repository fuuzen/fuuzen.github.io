# 正规子群

# 定义

设 HHH 是群 GGG 的子群 （⚠️得先是子群），如果 ∀a∈G∀a\in G∀a∈G 都有 Ha=aHHa=aHHa=aH ，则称 HHH 是 GGG 的正规子群 (normal subgroup) 或不变子群 (invariant subgroup)，记作:

H⊴GH\unlhd GH⊴G

条件 Ha=aHHa=aHHa=aH 仅表示两个集合 HaHaHa 与 aHaHaH 相等。不能错误地认为，由 Ha=aHHa=aHHa=aH 可推出对 HHH 的任意元素 hhh 有 ha=ahha=ahha=ah。

正确的理解是：对任意 h∈Hh\in Hh∈H ，存在 h′∈Hh^′\in Hh′∈H 使得 ha=ah′ha = ah'ha=ah′。

群 GGG 的单位元子群 {e}\{e\}{e} 和群 GGG 本身都是 GGG 的正规子群。这两个正规子群称为 GGG 的平凡正规子群。如果 GGG 只有平凡正规子群，且 G≠{e}G≠\{e\}G={e} ，则称 GGG 为单群 (simple group)

# 交换群的子群

显然交换群 GGG 的任意子群都是 GGG 的正规子群

# 等价定义

设 HHH 是群 GGG 的子群，如果 HHH 的任意一个左陪集也是它的一个右陪集，则 HHH 是 GGG 的正规子群

证明：

对任意 a∈Ga\in Ga∈G ，HHH 的左陪集 aHaHaH 也是 HHH 的右陪集，即存在 b∈Gb\in Gb∈G ，使得 aH=HbaH=HbaH=Hb ;

同时 a∈aHa\in aHa∈aH ，因此 a∈Hba\in Hba∈Hb ；

而又有 a∈Haa\in Haa∈Ha ，所以 a∈Ha∩Hba\in Ha\cap Hba∈Ha∩Hb ；

根据陪集的性质有 Ha=HbHa=HbHa=Hb ，因此 Ha=Hb=aHHa=Hb=aHHa=Hb=aH ，因此 HHH 是 GGG 的正规子群。

# 子集与正规子群

对于：

H⊆K⊆GH\subseteq K\subseteq GH⊆K⊆G

  • 若 HHH 是 GGG 的正规子群，则 HHH 也是 KKK 的正规子群
  • 若 HHH 是 KKK 的正规子群，且 KKK 是 GGG 的正规子群，则推不出 HHH 是 GGG 的正规子群！也就是正规子群关系 ⊴/⊵\unlhd/\unrhd⊴/⊵ 对子集关系不具有传递性⚠️
