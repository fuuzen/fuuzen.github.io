# 环的相关定义

设 RRR 是非空集合，如果在 RRR 上定义了两个二元运算 “+++”（称为加法）和 “ ⋅\cdot⋅ ”（称为乘法），且满足：

  • （1） RRR 关于加法 +++ 构成交换群（即加群）
  • （2） RRR 关于乘法 ⋅\cdot⋅ 构成半群，即乘法满足结合律：∀a,b,c∈R,a⋅(b⋅c)=(a⋅b)⋅c\forall a,b,c\in R,\ \ a\cdot(b\cdot c)=(a\cdot b)\cdot c∀a,b,c∈R, a⋅(b⋅c)=(a⋅b)⋅c；
  • （3） 乘法 ⋅\cdot⋅ 对加法有左右分配律：

∀a,b,c∈Ra⋅(b+c)=a⋅b+a⋅c(b+c)⋅a=b⋅a+c⋅a\forall a,b,c\in R\\ a\cdot(b+c)=a\cdot b+a\cdot c\\ (b+c)\cdot a=b\cdot a+c\cdot a∀a,b,c∈Ra⋅(b+c)=a⋅b+a⋅c(b+c)⋅a=b⋅a+c⋅a

则称代数 (R,+,⋅)(R,+,\ \cdot\ )(R,+, ⋅ ) 是环（ring）。在通过上下文能明确运算时，通常直接称 RRR 为环。

有教材要求环乘法必须有单位元，也有教材要求环乘法必须满足交换律等

对环 (R,+,⋅)(R,+,\ \cdot\ )(R,+, ⋅ ) ，(R,+)(R\ ,+)(R ,+) 是交换群，称为环 RRR 的加法群，其单位元通常用 0 表示，称为环 RRR 的零元，元素 aaa 的加法逆元记为 −a-a−a ，称为 aaa 的负元。

通过负元引入环的减法，即 a−ba-ba−b 定义为 a+(−b)a+(-b)a+(−b)。

  • 环乘法是否满足交换律
    • 若环 RRR 的乘法也满足交换律，则称为交换环 (commutative ring)
  • 是否有单位元
    • 若环 RRR 的乘法有单位元，则称为有单位元环，环乘法的单位元通常记为 eee 或 111 ，并称为环 RRR 的单位元
  • 环乘法当环 RRR 有单位元 eee 时，考虑环的元素是否有逆元
    • 若对 a∈Ra\in Ra∈R ，存在 b∈Rb\in Rb∈R ，使得 ab=ba=eab=ba=eab=ba=e ，则称 aaa 是可逆元，或称为环 RRR 的单位 (unit)，并称 bbb 是 aaa 的逆元。当然这时 bbb 也是可逆元，且 aaa 是 bbb 的逆元
    • 所有可逆元关于环乘法构成群，称为 RRR 的单位群 (group of units)，记为 U(R)U(R)U(R)。
  • 环乘法是否有零因子
    • 对环 RRR 的两个非零元素 a,ba,ba,b ，若 a⋅b=0a\cdot b=0a⋅b=0 ，则称 aaa 是左零因子 (left zero-divisor)，b 是右零因子 (right zero-divisor)。左零因子和右零因子统称为零因子
    • 若环 RRR 的所有非零元素都不是左零因子或右零因子，则称 RRR 为无零因子环。

当集合 RRR 只有一个元素 000 ，定义 RRR 的加法运算和乘法运算：0+0=0,0⋅0=00+0=0,\ 0\cdot 0=00+0=0, 0⋅0=0，则 ({0},+,⋅)(\{0\},+,\ \cdot\ )({0},+, ⋅ ) 构成环，这个环称为零环。在这个环中，零元也是单位元，而零元也是可逆元。

零环过于简单，通常在对环进行讨论时都将零环排除在外。因此在提到环时，总是默认环至少有两个元素，这时零元不可能是单位元，也不可能是可逆元。

# 整环

  • 有单位元
  • 无零因子
  • (乘法) 可交换

的环称为整环 (integral domain)

证明一个环是整环：

\forall a,b\in R,\ \ ab=0\Rightarrow a=0\or b=0

# 除环

  • 有单位元
  • 至少有两个元素
  • 每个非零元都可逆 (乘法)
  • （无零因子，可逆元都不是零因子）

的环称为除环 (division ring)

# 域

交换除环称为域 (field) (比较常用)，也即需满足：

  • 有单位元
  • 至少有两个元素
  • 每个非零元都可逆 (乘法)
  • (乘法) 可交换
  • （无零因子，可逆元都不是零因子）

可逆元都不是零因子（为什么？），因此域都是整环！

非交换除环也称体 (skew field) (比较少用)

# 理想

# 定义

设 RRR 是环，III 是 RRR 的非空子集。若 I 满足：

  1. ∀r1,r2∈I\forall r_1,r_2\in I∀r1​,r2​∈I，有 r1−r2∈Ir_1-r_2\in Ir1​−r2​∈I；
  2. ∀r∈I,∀s∈R,rs,sr∈I\forall r\in I,\ \forall s\in R,\ \ rs,sr\in I∀r∈I, ∀s∈R, rs,sr∈I；

则称 III 为环 RRR 的理想 (ideal)，记为 I⊲RI⊲RI⊲R，若 III 是 RRR 的真子集，则称 III 是 RRR 的真理想 (proper ideal)。

理想是子环：显然如果 III 是 RRR 的理想，则 III 必定是 RRR 的子环；

⚠️ 当然子环不一定是理想 ⚠️

RRR 的单个零元构成的集合 {0}\{0\}{0}（称为零理想）和 RRR 本身都是 RRR 的理想，这两个理想称为 RRR 的平凡理想。因此 RRR 的非平凡理想就是非零真理想

# 商环

环关于理想的商环

# 定义

设 RRR 是环，III 是 (R,+,⋅)(R,+,\ \cdot\ )(R,+, ⋅ ) 的一个理想，则 (I,+)(I,+)(I,+) 是 (R,+)(R,+)(R,+) 的正规子群，它所有（关于加群）的陪集构成集合 R/I＝{x+I∣x∈R}R/I＝\{x+I|x\in R\}R/I＝{x+I∣x∈R} 、记陪集 x+Ix+Ix+I 为 x‾\overline xx 。

# 运算的定义

# 加法

在集合 R/IR/IR/I 上可定义加法 +++：

∀x‾,y‾∈R/I,x‾+y‾=x+y‾⟺(x+I)+(y+I)=(x+y)+I\forall \overline x,\overline y\in R/I,\\ \overline{x}+\overline{y}=\overline{x+y}\Longleftrightarrow (x+I)+(y+I)=(x+y)+I∀x,y​∈R/I,x+y​=x+y​⟺(x+I)+(y+I)=(x+y)+I

这里 x‾+y‾\overline{x}+\overline{y}x+y​ 中的 +++ 是要定义的加法，即后面所说商环中的加法，而 x+y‾\overline{x+y}x+y​ 中的 +++ 是环 RRR 中的加法。

∀x‾,y‾∈R/I,x‾⋅y‾=x⋅y‾⟺(x+I)⋅(y+I)=(x⋅y)+I\forall \overline x,\overline y\in R/I,\\ \overline{x}\cdot\overline{y}=\overline{x\cdot y}\Longleftrightarrow (x+I)\cdot(y+I)=(x\cdot y)+I∀x,y​∈R/I,x⋅y​=x⋅y​⟺(x+I)⋅(y+I)=(x⋅y)+I

这里 x‾⋅y‾\overline{x}\cdot\overline{y}x⋅y​ 中的 $\cdot $ 是要定义的乘法，即后面所说商环中的乘法，而 x⋅y‾\overline{x\cdot y}x⋅y​ 中的 $\cdot $ 是环 RRR 中的乘法。

# 环同态

环的同态就是与环的加法与乘法都可交换的函数

设 (R,+,⋅)(R,+,\cdot)(R,+,⋅) 和 (R′,⊕,⊗)(R',\oplus,\otimes)(R′,⊕,⊗) 是环，φ:R→R′\varphi:R\to R'φ:R→R′ 是 RRR 到 R′R'R′ 的函数，若对 ∀a,b∈R\forall a,b\in R∀a,b∈R 有：

φ(a+b)=φ(a)⊕φ(b)φ(a⋅b)=φ(a)⊗φ(b)\varphi(a+b)=\varphi(a)\oplus\varphi(b)\\ \varphi(a\cdot b)=\varphi(a)\otimes\varphi(b)φ(a+b)=φ(a)⊕φ(b)φ(a⋅b)=φ(a)⊗φ(b)

则称 φ\varphiφ 是 RRR 到 R′R'R′ 的同态。

  • φ\varphiφ 是单函数，则称为单同态
  • φ\varphiφ 是满函数，则称为满同态
  • φ\varphiφ 是双函数，则称为同构，记为：R≅R′R\cong R'R≅R′。

设 RRR 和 R′R'R′ 是两个环，定义函数 φ:R→R′φ:R→R^′φ:R→R′：∀a∈R,φ(a)=0∀a\in R,φ(a)= 0∀a∈R,φ(a)=0，这里 0 是 R′R'R′ 的零元，则容易验证 φφφ 是同态，这个同态称为零同态 (zero homomorphism)

设 RRR 是环，III 是 RRR 的理想，则很自然地有同态 ρ:R→R/Iρ:R→R/Iρ:R→R/I ，∀a∈R,ρ(a)=a‾∀a\in R,ρ(a)=\overline a∀a∈R,ρ(a)=a，这里 a‾\overline aa 是 aaa 所在的等价类，即 a+Ia+Ia+I 。这个同态称为商环 R/IR/IR/I 的自然同态 (natural morphism)

零同态、自然同态都是满同态。

自然同态一个例子：

  • 整数环 Z\mathbb ZZ 到模 mmm 剩余类环 Zm\mathbb Z_mZm​ 有很自然的满同态 φ:Z→Zmφ:\mathbb Z→\mathbb Z_mφ:Z→Zm​，∀z∈Z,φ(z)=zmodm∀z\in \mathbb Z, φ(z)=z\bmod m∀z∈Z,φ(z)=zmodm

零同态全部映射到零元，单同态给出子代数，满同态给出商代数

环同态保持：

  • 加法单位元
  • 子环

# 环扩张

环的扩张定理，也称为挖补定理

如图，RRR 和 S′S'S′ 是 2 个没有公共元素的环，存在 RRR 到 S′S'S′ 的单同态：

ψ:R→S′\psi:R\to S'ψ:R→S′

并且：ψ(R)=S\psi(R)=Sψ(R)=S。

那么 RRR 可以扩张为 R′R'R′ ，满足：

  • R′−R=S′−SR'-R=S'-SR′−R=S′−S；
  • S′≅R′S'\cong R'S′≅R′；

设 RRR 是一个没有单位元的环，则存在一个有单位元的环 R′R'R′ ，使得 RRR 是 R′R'R′ 的子环。

# 特征

设 RRR 是环。若存在最小正整数 nnn 使得对所有 a∈Ra\in Ra∈R 有 na=0na=0na=0 ，则称 nnn 为环 RRR 的特征。如果不存在这样的正整数，则称 RRR 的特征为 0。环 R 的特征记为 Char RRR。

数域 Z,Q,R,C\mathbb Z, \mathbb Q, \mathbb R, \mathbb CZ,Q,R,C 的特征都是 0，而模 mmm 剩余类环 Zm\mathbb Z_mZm​ 的特征是 mmm。

【定理】设 RRR 是有单位元 eee 的环。若 eee 关于加法的阶为无穷大，则 Char R=0R=0R=0，否则 Char R=∣e∣R=|e|R=∣e∣，这里 ∣e∣|e|∣e∣ 在环的加群中的阶。

证明 若 eee 的加法的阶为无穷大，则不存在正整数 nnn 使得 ne=0ne=0ne=0 ，从而 Char R=0R= 0R=0，否则，若 ∣e∣=n|e|=n∣e∣=n ，则 ne=0ne=0ne=0 ，从而对任意 a∈Ra\in Ra∈R ，na=n(ea)=(ne)a=0na=n(ea)=(ne)a=0na=n(ea)=(ne)a=0。

整环的特征是 0 或是一个素数，进而域的特征也只能是 0 或是素数。

根据 RRR 的特征构造 Z\mathbb ZZ 到 RRR 的同态

【定理】设 RRR 是有单位元 eee 的环，定义函数 φ:Z→Rφ:\mathbb Z→Rφ:Z→R，∀n∈Z,φ(n)=ne∀n\in \mathbb Z, φ(n)=ne∀n∈Z,φ(n)=ne，则 φφφ 是环 Z\mathbb ZZ 到 RRR 的同态。

【推论】设 RRR 是有单位元 eee 的环。

  1. 如果 RRR 的特征为 n>0n>0n>0，则 RRR 包含一个与 Zn\mathbb Z_nZn​ 同构的子环；

  2. 如果 RRR 的特征为 0 ，则 RRR 包含一个与 Z\mathbb ZZ 同构的子环。

    • 所以特征为 0 的环一定是无穷环

⭐️每个有限域的阶必为素数的幂，即有限域的阶可表示为 pnp^npn ( ppp 是素数、nnn 是正整数)，该有限域通常称为 Galois 域 (Galois Fields)，记为 GF(pn)GF(p^n)GF(pn)。

【推论】设 FFF 是域。

1. 如果 FFF 的特征为 0 ，则 FFF 包含一个与有理数域同构的子域；

2. 如果 RRR 的特征是素数 ppp ，则 FFF 包含一个与模 ppp 剩余类环 Zp\mathbb Z_pZp​ 同构的子域。

# 素域

若域 FFF 不包含任何真子域，则称 F 是素域 (prime field)。

特征为素数 ppp 的交换环中：

(a±b)pn=apn+bpn(a\pm b)^{p^n}=a^{p^n}+b^{p^n}(a±b)pn=apn+bpn

证明思路

交换环中也有二项式定理，以此证明 (a±b)p=ap+bp(a\pm b)^{p}=a^{p}+b^{p}(a±b)p=ap+bp ，然后数学归纳法即可。
