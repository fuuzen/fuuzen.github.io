# 群 (Group) 的定义

如果集合 G 及其二元运算 ⋅\cdot⋅ 满足下面的性质，则称 (G, ⋅\cdot⋅ ) 是群：

  • 二元运算 ⋅\cdot⋅ 满足结合律：即∀a,b,c∈G, (a ⋅\cdot⋅ b) ⋅\cdot⋅ c = a ⋅\cdot⋅ (b ⋅\cdot⋅ c)
  • 有单位元 e，即存在 e∈G，使得∀a∈G, e ⋅\cdot⋅ a=a ⋅\cdot⋅ e=a
  • ∀a∈G 关于 ⋅\cdot⋅ 有逆元a−1a^{-1}a−1，即∀a∈G，存在 b∈G，使得 a ⋅\cdot⋅ b = b ⋅\cdot⋅ a = e，记 b 为a^

从代数系统角度说，群有三个运算，二元运算 ⋅\cdot⋅ ，零元运算 e 和一元运算(-)^

  • 其中二元运算占主导地位，在谈到群时经常只明确给出它的二元运算
  • 进一步，在上下文能明确其运算时，人们通常直接称集合 G 是群

如果集合 S 有满足结合律的二元运算 ⋅\cdot⋅ ，则称 (S, ⋅\cdot⋅ ) 是半群 (semi-group)

如果 ⋅\cdot⋅ 还有单位元 e，则称 (S, ⋅\cdot⋅ , e) 是独异点 (monoid)

# 常用群举例

# 整数加群

整数集 Z 及整数加法运算构成群，称为整数加群

  • 单位元是 0
  • 每个整数 z 的逆元是 - z

# 模 m 剩余类加群

对整数 m≥2，记 Z_m={0,1,⋯, m-1}，Z_m 及 模 m 加 ⊕m⊕_m⊕m​ 构成群，称为模 m 剩余类加群

0 是单位元，对每个整数 z∈Z_m，它的逆元是 ((-z)+m) mod m

# Q∗\mathbb Q^*Q∗ R∗\mathbb R^*R∗ C∗\mathbb C^*C∗ 乘法群

整数集 Z\mathbb ZZ 关于整数乘法不构成群，但非零有理数集Q∗\mathbb Q^∗Q∗、非零实数集 R∗\mathbb R^∗R∗、非零复数集 C∗\mathbb C^*C∗ 关于数的乘法运算构成群

# 实矩阵乘法群

记实数域上的 n 阶方阵构成的集合为 Mn(R)M_n (\mathbb R)Mn​(R)，则它关于矩阵加法构成群；记实数域上的 n 阶可逆方阵构成的集合为 GLn(R)GL_n (\mathbb R)GLn​(R)，则它关于矩阵乘法构成群

# 双函数复合群

集合 S 上的所有双函数 f:S→Sf:S→Sf:S→S 及复合构成群

  • 恒等函数是单位元
  • 逆函数给出逆元

# 群的一些术语

# 交换群

群的运算不一定满足交换律，满足交换律的群称为交换群，也称为阿贝尔群

# 群的阶

群 G 的元素个数（准确地说 G 的基数）|G | 称为群 G 的阶 (order)，如果 G 是有穷集，则称为有穷群（有限群），否则称为无穷群（无限群）

# 加群

当群称为加群时通常将它：

  • 单位元记为 0
  • 元素 a 的逆元记为 - a
  • 运算称为加法
  • 运算的结果称为和
  • 运算用加号 + 表示

通常只有当群是交换群的时候，才使用加号 '+' 表示这个群的运算

# 乘群

将不是加群的群称为乘群，并将：

  • 乘群的单位元通常用 1 或 e 表示
  • 元素 a 的逆元用a−1a^{-1}a−1 表示
  • 运算称为乘法
  • 运算结果称为积
  • 运算用 ∘\circ∘ 或 ⋅\cdot⋅ 或 ∗ 表示，但通常省略不写！

# 群的一些基本性质

群有单位元，因此群不可能是空集

群要求每个元素都有逆元，而运算的零元不可能有逆，因此群没有零元（除平凡群 ({e}, ∘\circ∘) 外）！

# 定理 1

设 G 是群，有：

∀a∈G，(a−1)−1=a∀a∈G，(a^{-1})^{-1}= a∀a∈G，(a−1)−1=a

∀a,b∈G,(ab)−1=b−1a−1∀a, b∈G, (ab)^{-1}= b^{-1} a^{-1}∀a,b∈G,(ab)−1=b−1a−1

【证明】

根据逆元的定义有 a 和 a−1a^{-1}a−1 互为逆元。

(b^{-1} a^{-1})(ab)=b^{-1}(a^{-1}a)b=b^{-1}eb=b^{-1} b=e$

类似地:

ab(b−1a−1)=eab(b^{-1}a^{-1})=eab(b−1a−1)=e

因此 ababab 与 b−1a−1b^{-1}a^{-1}b−1a−1 互为逆元。

# 定理 2

群的二元运算满足消去律：

设 G 是群，对任意 a,b,c∈Ga,b,c∈Ga,b,c∈G, ab=acab=acab=ac 蕴涵 b=cb=cb=c ，同样 ba=caba=caba=ca 蕴涵 b=cb=cb=c

【证明】

若 ab=acab=acab=ac，则有 a−1(ab)=a−1(ac)a^{-1}(ab)=a^{-1} (ac)a−1(ab)=a−1(ac)

而群运算满足结合律，因此 (a−1a)b=(a−1a)c(a^{-1} a)b=(a^{-1} a)c(a−1a)b=(a−1a)c

而a−1a=ea^{-1} a=ea−1a=e，e 是单位元，从而有 b=c

同理可证 ba=ca 蕴涵 b=c

# 定理 3

给定非空集 G 及其上的二元运算（假定是乘法运算而省略其运算符号）。G 关于该二元运算构成群的充分必要条件是：

  • G 的二元运算满足结合律
  • G 的二元运算有左单位元：即存在 e∈Ge\in Ge∈G 使得对任意的 a∈Ga\in Ga∈G 有 ea =a

# 定理 4

给定非空集 G 及其二元运算，且该二元运算满足结合律。G 构成群的充分必要条件是：

对任意的 a,b∈Ga,b\in Ga,b∈G，方程 ax=bax=bax=b 与 ya=bya=bya=b 在 G 中都有解。

# 问题 5

  • 该运算有左单位元 e，即 ∀a∈G 有 ea=a
  • 对关于该左单位元，每个元素有右逆，即任意 a∈G，存在 b 使得 ab=e
