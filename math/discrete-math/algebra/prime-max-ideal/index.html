# 素理想

非交换环上也可以定义素理想、极大理想，但这里只考虑交换环上的素理想和极大理想！

# 定义

设 RRR 是交换环，PPP 是 RRR 的真理想。若对任意 a,b∈Ra,b\in Ra,b∈R，ab∈Pab\in Pab∈P 蕴涵 a∈Pa\in Pa∈P 或 b∈Pb\in Pb∈P ，则称 PPP 为 RRR 的素理想 (prime ideal)。

素理想的概念 b 本质上来自数论中的基本结论：若 ppp 为素数，则 p|ab\Rightarrow p|a\or p|b。

# 整数环的素理想

设 nnn 是正整数，⟨n⟩\langle n\rangle⟨n⟩ 是整数环 Z\mathbb ZZ 的素理想当且仅当 nnn 是素数。

零理想 I={0}I=\{0\}I={0} 是整数环 Z\mathbb ZZ 的素理想，但是 Z\mathbb ZZ 不是素理想，因为素理想要求是真理想！

# 素理想的充要条件

设 RRR 是有单位元 e≠0e≠0e=0 的交换环，III 是 RRR 的理想，则 III 是 RRR 的素理想当且仅当 R/IR/IR/I 是整环。

# 极大理想

# 定义

设 RRR 是交换环，MMM 是 RRR 的真理想。若对 RRR 的任意包含 MMM 的理想 NNN ，必有 N=MN=MN=M 或 N=RN=RN=R，则称 MMM 是 RRR 的极大理想 (maximal ideal)。

极大理想就是没有真包含它的非平凡理想

# 极大理想的证明

证明 III 是 RRR 的极大理想的思路：（RRR 是有单位元 eee 的交换环）

假设 JJJ 是理想且 I⊂JI\sub JI⊂J，...... 得到 e∈Je\in Je∈J，所以 J=RJ=RJ=R，所以 III 是 RRR 的极大理想。

# 整数环的极大理想

设 ppp 是正整数，⟨p⟩\langle p\rangle⟨p⟩ 是整数环 Z\mathbb ZZ 的极大理想当且仅当 ppp 是素数，和素理想一致！

# 极大理想的充要条件

设 RRR 是有单位元 eee 的交换环，III 是 RRR 的理想，则：III 是 RRR 的极大理想当且仅当 R/IR/IR/I 是域。

  • 在交换环 RRR 中，III 是 RRR 的素理想当且仅当在 R/IR/IR/I 中 aˉbˉ=0ˉ\bar{a}\bar{b}=\bar0aˉbˉ=0ˉ 蕴涵 aˉ=0ˉ\bar a=\bar 0aˉ=0ˉ 或 bˉ=0ˉ\bar b=\bar 0bˉ=0ˉ，当且仅当 R/IR/IR/I 的每个元素不是零因子
  • 在有单位元交换环 RRR 中，III 是 RRR 的极大理想当且仅当每个真包含 III 的理想都包含 RRR 的单位元（从而就包含 RRR 的所有元素）

# 推论

设 RRR 是有单位元的交换环，由于域都是整环，因此 RRR 的每个极大理想都是 RRR 的素理想。

  • 如果没有单位元，则极大理想不一定是素理想
    • 例如对于 R=2ZR=2\mathbb ZR=2Z，I=4ZI=4\mathbb ZI=4Z，III 是 RRR 的极大理想，但 III 不是 RRR 的素理想
  • 一个素理想（即使是非零素理想）也不一定是极大理想

# 多项式环的极大理想

考察商环 Zm[x]/⟨ax2+bx+c⟩\mathbb Z_m[x]/\langle ax^2+bx+c\rangleZm​[x]/⟨ax2+bx+c⟩ 是否是域，可以看其所有非零元素是否构成循环群，若构成循环群，则每个非零元素都可逆，则商环是域。

# 高斯整环的极大理想

以下结论包括了高斯整环中的所有极大理想：

  • 高斯整环 Z[i]\mathbb Z[i]Z[i] 的主理想 ⟨a+bi⟩\langle a+bi\rangle⟨a+bi⟩ 是极大理想当且仅当 a2+b2a^2+b^2a2+b2 是素数，且高斯整环 Z[i]\mathbb Z[i]Z[i] 关于极大理想 ⟨a+bi⟩\langle a+bi\rangle⟨a+bi⟩ 的商环 Z[i]/⟨a+bi⟩\mathbb Z[i]/\langle a+bi\rangleZ[i]/⟨a+bi⟩ 和模 a2+b2a^2+b^2a2+b2 剩余类环 Za2+b2\mathbb Z_{a^2+b^2}Za2+b2​ 同构。

a2+b2a^2+b^2a2+b2 是素数，则一定有 a2+b2≡3mod4a^2+b^2\equiv3\bmod4a2+b2≡3mod4

  • 高斯整环 Z[i]\mathbb Z[i]Z[i] 的主理想 ⟨p⟩\langle p\rangle⟨p⟩ 是极大理想当且仅当 ppp 是素数并且 p≡3mod4p\equiv3\bmod4p≡3mod4，和整数环中一致。且高斯整环 Z[i]\mathbb Z[i]Z[i] 关于极大理想 ⟨p⟩\langle p\rangle⟨p⟩ 的商环 Z[i]/⟨p⟩\mathbb Z[i]/\langle p\rangleZ[i]/⟨p⟩ 和模 p2p^2p2 剩余类环 Zp2\mathbb Z_{p^2}Zp2​ 同构。

  • 高斯整环中，⟨1+i⟩\langle1+i\rangle⟨1+i⟩ 和 ⟨1−i⟩\langle1-i\rangle⟨1−i⟩ 是极大理想。
