# 函数

(全函数)

# 描述

集合A到B的函数，是A×B笛卡尔积的一类子集集合A到A的函数也称为A上的函数定义域的每个元素都存在陪域的唯一元素与之对应\begin{aligned} &集合A到B的函数，是A\times B笛卡尔积的一类子集\\ &集合A到A的函数也称为A上的函数\\ &定义域的每个元素都存在陪域的唯一元素与之对应 \end{aligned}​集合A到B的函数，是A×B笛卡尔积的一类子集集合A到A的函数也称为A上的函数定义域的每个元素都存在陪域的唯一元素与之对应​
