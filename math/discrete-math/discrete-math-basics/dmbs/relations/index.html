# 笛卡尔积

# 定义

A×B={<a,b>∣a∈A∧b∈B}\begin{aligned}& A\times B=\{<a,b>|a\in A\land b\in B\} \end{aligned}​A×B={<a,b>∣a∈A∧b∈B}​

# 性质

笛卡尔积运算不满足交换律\begin{aligned}& 笛卡尔积运算不满足交换律 \end{aligned}​笛卡尔积运算不满足交换律​

笛卡尔积运算不满足结合律\begin{aligned}& 笛卡尔积运算不满足结合律 \end{aligned}​笛卡尔积运算不满足结合律​

笛卡尔积运算对集合交有分配律\begin{aligned}& 笛卡尔积运算对集合交有分配律 \end{aligned}​笛卡尔积运算对集合交有分配律​

# 二元关系

# 描述

集合A到B的二元关系是笛卡尔积A×B的子集\begin{aligned}& 集合A到B的二元关系是\\& 笛卡尔积A\times B的子集 \end{aligned}​集合A到B的二元关系是笛卡尔积A×B的子集​

集合A上的二元关系是集合A到自己的二元关系\begin{aligned}& 集合A上的二元关系是\\& 集合A到自己的二元关系 \end{aligned}​集合A上的二元关系是集合A到自己的二元关系​

# 定义

R⊆A×B\begin{aligned}& R\subseteq A\times B \end{aligned}​R⊆A×B​

R⊆A×A\begin{aligned}& R\subseteq A\times A \end{aligned}​R⊆A×A​

# 描述

a和b有关系R\begin{aligned}& a和b有关系R \end{aligned}​a和b有关系R​

aRb\begin{aligned}& a\ R\ b \end{aligned}​a R b​

<a,b>∈R\begin{aligned}& <a,b>\in R \end{aligned}​<a,b>∈R​

a和b没有关系R\begin{aligned}& a和b没有关系R \end{aligned}​a和b没有关系R​

aR̸b\begin{aligned}& a\ \not R\ b \end{aligned}​a R b​

<a,b>∉R\begin{aligned}& <a,b>\not\in R \end{aligned}​<a,b>∈R​

# 备注

本课程的关系只考察二元关系\begin{aligned}& 本课程的关系只考察二元关系 \end{aligned}​本课程的关系只考察二元关系​

# 恒等关系

or
对角关系

# 定义

ΔA={<a,a>∣a∈A},A≠∅\begin{aligned}& \Delta_A=\{<a,a>|a\in A\},\ \ A\neq \varnothing \end{aligned}​ΔA​={<a,a>∣a∈A}, A=∅​

# 关系矩阵

单位矩阵\begin{aligned}& 单位矩阵 \end{aligned}​单位矩阵​

# 自反关系

# 定义

∀a∈A,<a,a>∈R\begin{aligned}& \forall a\in A,\ <a,a>\in R \end{aligned}​∀a∈A, <a,a>∈R​

# 关系矩阵

ΔA⊆R\begin{aligned}& \Delta_A\subseteq R \end{aligned}​ΔA​⊆R​

# 反自反关系

# 定义

∀a∈A,<a,a>∉R\begin{aligned}& \forall a\in A,\ <a,a>\not\in R \end{aligned}​∀a∈A, <a,a>∈R​

# 关系矩阵

ΔA∩R=∅\begin{aligned}& \Delta_A \cap R=\varnothing \end{aligned}​ΔA​∩R=∅​

# 对称关系

# 定义

∀a,b∈A,<a,b>∈R→<b,a>∈R\begin{aligned}& \forall a,b\in A,\ <a,b>\in R\to<b,a>\in R \end{aligned}​∀a,b∈A, <a,b>∈R→<b,a>∈R​

# 关系矩阵

R=R−1\begin{aligned}& R=R^{-1} \end{aligned}​R=R−1​

# 反对称关系

# 定义

∀a,b∈A,(<a,b>∈R)∧(<b,a>∈R)→a=b\begin{aligned}& \forall a,b\in A,\ (<a,b>\in R)\land(<b,a>\in R)\to a=b \end{aligned}​∀a,b∈A, (<a,b>∈R)∧(<b,a>∈R)→a=b​

# 关系矩阵

R∩R−1⊆ΔA\begin{aligned}& R\cap R^{-1}\subseteq\Delta_A \end{aligned}​R∩R−1⊆ΔA​​

# 备注

反对称关系一定是对称关系\begin{aligned}& 反对称关系一定是对称关系 \end{aligned}​反对称关系一定是对称关系​

# 传递关系

# 定义

∀a,b,c∈A,(<a,b>∈R)∧(<b,c>∈R)→<a,c>∈R\begin{aligned}& \forall a,b,c\in A,\ (<a,b>\in R)\land(<b,c>\in R)\to <a,c>\in R \end{aligned}​∀a,b,c∈A, (<a,b>∈R)∧(<b,c>∈R)→<a,c>∈R​

# 关系矩阵

R∘R−1⊆R\begin{aligned}& R\circ R^{-1}\subseteq R \end{aligned}​R∘R−1⊆R​

# 等价关系

# 定义

R是自反的、对称的、传递的\begin{aligned}& R是自反的、对称的、传递的 \end{aligned}​R是自反的、对称的、传递的​

# 等价类

  • 描述

a所在的等价关系R的等价类，简称a的等价类\begin{aligned}& a所在的等价关系R的等价类，简称a的等价类 \end{aligned}​a所在的等价关系R的等价类，简称a的等价类​

  • 定义

[a]R={x∈A∣<a,x>∈R}\begin{aligned}& [a]_R=\{x\in A|<a,x>\in R\} \end{aligned}​[a]R​={x∈A∣<a,x>∈R}​

# 代表

b∈[a]R，则称b为等价类[a]R的一个代表\begin{aligned}& b\in [a]_R，则称b为等价类[a]_R的一个代表 \end{aligned}​b∈[a]R​，则称b为等价类[a]R​的一个代表​

# 商集

  • 描述

集合A关于等价关系R的商集是集合A的所有等价类构成的集合\begin{aligned}& 集合A关于等价关系R的商集是\\& 集合A的所有等价类构成的集合 \end{aligned}​集合A关于等价关系R的商集是集合A的所有等价类构成的集合​

  • 定义

A/R={[a]R∣a∈A}\begin{aligned}& A/R=\{[a]_R|a\in A\} \end{aligned}​A/R={[a]R​∣a∈A}​

  • 基本性质

商集是一个集合族且是A的划分其中每一个元素都是等价类每个等价类都是这个划分的一个划分块\begin{aligned}& 商集是一个集合族且是A的划分\\& 其中每一个元素都是等价类\\& 每个等价类都是这个划分的一个划分块 \end{aligned}​商集是一个集合族且是A的划分其中每一个元素都是等价类每个等价类都是这个划分的一个划分块​

# 偏序关系

# 定义

R是自反的、反对称的、传递的\begin{aligned}& R是自反的、反对称的、传递的 \end{aligned}​R是自反的、反对称的、传递的​

# 符号

⪯\begin{aligned}& \preceq \end{aligned}​⪯​

# 经典例子

  • 数集上的 小于或等于关系、大于或等于关系

    • 集合的子集关系

      • 整数集上的整除关系

# 可比

(a⪯b)∨(b⪯a)\begin{aligned}& (a\preceq b) \lor (b\preceq a) \end{aligned}​(a⪯b)∨(b⪯a)​

# 不可比

(a⪯̸b)∧(b⪯̸a)\begin{aligned}& (a\not\preceq b) \land (b\not\preceq a) \end{aligned}​(a⪯b)∧(b⪯a)​

# 覆盖

b覆盖a⇔(a⪯b)∧!∃c(a⪯c∧c⪯b)\begin{aligned}& b覆盖a\Leftrightarrow(a\preceq b)\land!\exists c(a\preceq c\land c\preceq b) \end{aligned}​b覆盖a⇔(a⪯b)∧!∃c(a⪯c∧c⪯b)​

# 极大元

a是A的极大元⇔∀b∈A(a⪯b→b=a)\begin{aligned}& a是A的极大元\Leftrightarrow \forall b\in A(a\preceq b\to b=a) \end{aligned}​a是A的极大元⇔∀b∈A(a⪯b→b=a)​

# 极小元

a是A的极小元⇔∀b∈A(b⪯a→b=a)\begin{aligned}& a是A的极小元\Leftrightarrow \forall b\in A(b\preceq a\to b=a) \end{aligned}​a是A的极小元⇔∀b∈A(b⪯a→b=a)​

# 最大元

a是A的最大元⇔∀b∈A(b⪯a)\begin{aligned}& a是A的最大元\Leftrightarrow \forall b\in A(b\preceq a) \end{aligned}​a是A的最大元⇔∀b∈A(b⪯a)​

# 最小元

a是A的最小元⇔∀b∈A(a⪯b)\begin{aligned}& a是A的最小元\Leftrightarrow \forall b\in A(a\preceq b) \end{aligned}​a是A的最小元⇔∀b∈A(a⪯b)​

# 偏序集

# 定义

(A,⪯)orabbreviatedasA\begin{aligned}& (A,\preceq)\ \ or\ \ abbreviated\ as\ A \end{aligned}​(A,⪯) or abbreviated as A​

# 全序

or
线序

# 定义

偏序集A的任意两个元素都可比，则这个偏序集是全序或线序\begin{aligned}& 偏序集A的任意两个元素都可比，则这个偏序集是全序或线序 \end{aligned}​偏序集A的任意两个元素都可比，则这个偏序集是全序或线序​
