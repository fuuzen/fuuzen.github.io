# 基本概念

# 不严格定义的概念

# 集合

作为整体研究的一堆东西，用大写字母 A, B, C, ⋯表示

# 元素

集合这一堆东西中的每一个，用小写字母 a, b, c, ⋯表示

# 属于

元素与集合间的关系

  • 元素 a 属于集合 A ，记为 a∈Aa\in Aa∈A
  • a 不属于 A，记为 a∉Aa\notin Aa∈/A

元素与集合间的属于关系也称为成员关系，元素是集合的成员

# 全集

研究范围内的所有东西，记为 U

# 用逻辑语言严格定义的概念

# 子集关系

A⊆B⇔∀x(x∈A→x∈B)A \subseteq B \lrArr \forall x(x\in A \to x\in B)A⊆B⇔∀x(x∈A→x∈B)

# 集合相等

A=B⇔∀x(x∈A↔x∈B)A = B \lrArr \forall x(x\in A \lrarr x\in B)A=B⇔∀x(x∈A↔x∈B)

或者

A=B⇔A⊆B∧B⊆AA=B \lrArr A\subseteq B \land B \subseteq AA=B⇔A⊆B∧B⊆A

# 空集

∀x(x∉∅)\forall x(x\notin \varnothing)∀x(x∈/∅)

# 朴素集合论

# 外延原则

两个集合只要有完全相同的元素则是相等的集合，不考虑集合名字本身的内涵

# 概念 (名字) 的外延

概念 (名字) 的外延是它所指称的对象，内涵是它有区别于其他概念的属性全体

# 集合 (名字) 的外延

对于集合 (名字)，外延是它包含的所有元素，内涵则视具体的应用而定

# 定义集合的方法

# 元素枚举法

将集合的所有元素一一罗列出来

  • 适合元素比较少，或可按明显规律罗列元素时定义集合
  • 元素罗列规律明显时可使用省略号

# 性质概括法

用谓词概括一个集合的所有元素满足的共同性质

# 基本形式

A={x∣P(x)}A = \{x∣P(x)\}A={x∣P(x)}

含义是:

∀x(x∈A↔P(x))\forall x(x\in A \lrarr P(x))∀x(x∈A↔P(x))

允许 P 是任意性质时有可能产生悖论：罗素悖论

公理集合论运用子集分离原则避免悖论：

A={x∈B∣P(x)}A=\{x\in B∣P(x)\}A={x∈B∣P(x)}

B 是已知的大集合

# 扩展形式

A={f(y)∣P(y)}A=\{f(y)∣P(y)\}A={f(y)∣P(y)}

含义是

∀x(x∈A↔∃y(x=f(y)∧P(y)))\forall x(x∈A↔∃y(x=f(y)∧P(y)))∀x(x∈A↔∃y(x=f(y)∧P(y)))

这里 f 是一个函数，或说 f (x) 是含有自由变量 x 的表达式

# 归纳定义法

给出基本元素和从已有元素构造其他元素的规则

从某种意义上说，集合的归纳定义给出了构造集合元素的算法

# 集合的划分 (partition)

# 定义

设 A 是非空集合，F 是集合族，其中每个集合都是 A 的子集。说集合族 F 是 A 的划分，如果：

  • 非空：对任意的 S∈FS\in \mathcal{F}S∈F , S\neq \mathcal
  • 两两不交：对任意两个集合 S1,S2∈FS_1, S_2\in FS1​,S2​∈F , S1∩S2=∅S_1\cap S_2=\varnothingS1​∩S2​=∅
  • 覆盖集合 A：∪F=A\cup \mathcal{F}= A∪F=A

非空集合 A 的划分 F\mathcal{F}F 中的每个集合称为这个划分的一个 划分块 (block)

# 非空集合 A 上的等价关系与它的划分有一一对应关系

A 关于一个等价关系的商集是 A 的划分，而 A 的一个划分导出的 “在同一划分块” 关系是等价关系

进一步，A 关于 “在同一划分块” 这个等价关系的商集就是这个划分，而 A 关于等价关系的商集作为 A 的划分所导出的 “在同一划分块” 关系就是这个等价关系本身

# 模 m 剩余类

是一种常用等价类

对任意整数 a∈Za\in \mathbb{Z}a∈Z , a 在 R 下的等价类 [a]R[a]_R[a]R​ 称为整数集 Z\mathbb ZZ 的一个（与 a 同余的）模 m 剩余类，并记为:

a‾={x∈Z∣x≡a(modm)}={x∈Z∣m∣x−a}={a+mz∣z∈Z}\overline{a}=\{x\in \mathbb Z ∣ x ≡ a(\mod m) \}=\{x\in \mathbb Z∣m∣x-a\}=\{a+mz∣z\in \mathbb Z \}a={x∈Z∣x≡a(modm)}={x∈Z∣m∣x−a}={a+mz∣z∈Z}

# 模 m 剩余类的商集

Zm={0‾,1‾,...,m−1‾}={0,1,...,m−1}\mathbb{Z}_m = \{ \overline 0,\overline 1,...,\overline{m-1} \} = \{ 0,1,...,m-1 \}Zm​={0,1,...,m−1​}={0,1,...,m−1}
