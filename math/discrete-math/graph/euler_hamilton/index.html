# 欧拉图

# 定义

欧拉图存在欧拉闭迹，半欧拉图只存在欧拉迹

若能遍历完所有的边但是没法回到起始点，称为非欧拉图但是有欧拉迹

# 欧拉图

假定 GGG 是一个连通图，且是欧拉图，则下列命题等价：

  • GGG 是欧拉图
  • GGG 的每个点的度数是偶数
  • GGG 的边集能划分为边不重的圈的并
  • （GGG 存在欧拉闭迹）

# 半欧拉图

假定 GGG 是一个连通图，且是半欧拉图，则下列命题等价：

  • GGG 是半欧拉图
  • GGG 有且仅有 2 个点的度数是奇数
  • （GGG 存在欧拉迹）

# 小结论💡

  • 当 nnn 满足什么条件时，完全图 KnK_nKn​ 是欧拉图？ nnn 为奇数
  • 当 nnn 满足什么条件时，nnn 方体 QnQ_nQn​ 是欧拉图？ nnn 为偶数
  • 若完全二部图 Ka,bK_{a,b}Ka,b​ 为欧拉图，aaa 和 bbb 需满足什么条件？ a,ba,ba,b 均为偶数
  • 假设图 GGG 恰好有两个连通分支，并且每个连通分支都是欧拉图，若要将 GGG 变为欧拉图，最少需要添加几条边？ 最少需要添加 2 条边
  • 欧拉图、半欧拉图是否存在割边？ 欧拉图不存在割边，半欧拉图存在割边

弗勒里算法解决了在欧拉图中求出一条具体欧拉环游的方法，核心思想是尽可能避割边行走

# 欧拉图中扩展欧拉环游充要条件

设 GGG 是非平凡的欧拉图，且 v∈V(G)v\in V(G)v∈V(G) ，则 GGG 的每条具有起点 vvv 的迹都能扩展成 GGG 的欧拉环游当且仅当 G−vG-vG−v 是森林。

# 中国邮递员问题

1962，中国数学家管梅谷提出并解决了 “中国邮路问题”

即最优环游问题：在一个连通具有非负权的赋权图 GGG 中找到一条包含每条边（允许重复）且边权之和最小的闭途径，称之为最优 (欧拉) 环游

  • 若图 GGG 是一个欧拉图，则找出 GGG 的欧拉环游即可
  • 对一般图，其解法为：添加重复边以使 GGG 成为欧拉图 G∗G^*G∗ ，并使添加的重复边的边权之和为最小，再求 G∗G^*G∗ 的欧拉回路

管梅谷的结论：

假定 G∗G^*G∗ 是在图 GGG 中添加一些重复边得到的欧拉图，则 G∗G^*G∗ 具有最小权值的充要条件是：

  • GGG 的每一条边最多被添加一次
  • 对于 G∗G^*G∗ 的每个圈来说，该圈新添加的边的总权值不超过该圈总权值的一半

证明：必要性

若 GGG 中某条边在 G∗G^*G∗ 中被添加的次数超过 1，则去掉其中两条重复的边，我们将得到一个总权值更小，且以 GGG 为生成子图的欧拉图

上述与 “ G∗G^*G∗ 总权值最小” 相矛盾，因此每条边最多被添加一次

假定在 G∗G^*G∗ 中存在某个圈使得该圈新添加的边的总权值大于该圈权值的一半，不妨设为 CCC

那么在 G∗G^*G∗ 中，将 CCC 上新添加的边全部去掉，然后将原圈未添加新边的边都添加一条重复边

这样的操作使得该圈所有点的度数不变或改变 2，相应的图仍然是欧拉图，但是权值更小，这与 “ G∗G^*G∗ 总权值最小” 相矛盾

这个必要性的证明给出了最优欧拉环游的构造方法

# 半欧拉图最优欧拉环游

一个非负权的边赋权图 GGG 中只有两个奇度顶点 uuu 与 vvv , 设计一个求其最优欧拉环游的算法如下：

  • (1) 在 uuu 与 vvv 间求出一条最短路 PPP； (最短路算法)
  • (2) 在最短路 PPP 上，给每条边添加一条重复边得 GGG 的欧拉图 G∗G^*G∗；
  • (3) 在 GGG 的欧拉图 G∗G^*G∗ 中用 Fleury 算法求出一条欧拉环游；

证明

设 uuu 与 vvv 是 GGG 的两个奇度顶点，G∗G^*G∗ 是 GGG 的任意一个欧拉图。

考虑 G∗[E∗−E]G^*[E^*-E]G∗[E∗−E]， 显然它只有两个奇数顶点 uuu 与 vvv， 当然它们必须在 G∗[E∗−E]G^*[E^*-E]G∗[E∗−E] 的同一个分支中，因此，存在 (u,v)(u,v)(u,v) 路 P∗P^*P∗：

# 哈密顿图

# 定义

# 哈姆顿图

如果经过图 GGG 的每个顶点恰好一次后能够回到出发点，称这样的图为哈密尔顿图，简称 HHH 图。所经过的闭途径是 GGG 的一个生成圈，称为 GGG 的哈密尔顿圈，简称 HHH 圈

# 半哈密顿图

不需回到出发点

如果存在经过 GGG 的每个顶点恰好一次的路，称这样的图为半哈密尔顿图，称该路为 GGG 的哈密尔顿路，简称 HHH 路

# 小结论💡

  • 当 n≥3n\geq3n≥3 时，完全图 KnK_nKn​ 是否为哈密尔顿图？ 是
  • 当 n≥2n\geq2n≥2 时， 方体 QnQ_nQn​ 是否为哈密尔顿图？ 是
  • 若完全二部图 Ka,bK_{a,b}Ka,b​ 为哈密尔顿图，aaa 和 bbb 需满足什么条件？⇒a=b≥2\Rightarrow a=b\geq2⇒a=b≥2！
  • 是否存在一个具有奇数个顶点的连通图，它既是二部图，也是哈密尔顿图？ 不存在，否则二部图中出现了奇圈
  • 若二部图是哈密尔顿图，则它的二部划分 (X,Y)(X,Y)(X,Y) 满足什么条件？⇒∣X∣=∣Y∣\Rightarrow|X|=|Y|⇒∣X∣=∣Y∣！

若 GGG 是 HHH 图，则对于 VVV 的每个非空真子集 SSS，均有：

ω(G−S)≤∣S∣\omega(G-S)\leq|S|ω(G−S)≤∣S∣

# 充分条件

# Dirac

对于 n≥3n≥3n≥3 的简单图 GGG , 如果 GGG 中有：

δ(G)≥n2\delta(G)\geq\frac{n}{2}δ(G)≥2n​

那么 GGG 是 HHH 图。

# Ore

对于 n≥3n≥3n≥3 的单图 GGG , 如果 GGG 中的任意两个不相邻顶点 uuu 与 vvv ，有：

d(u)+d(v)≥nd(u)+d(v)\geq nd(u)+d(v)≥n

那么 GGG 是 HHH 图。

该定理证明和 Dirac 定理可以完全一致！

该定理的条件是紧的。例如：设 GGG 是由 Kk+1Κ_{k+1}Kk+1​ 的一个顶点和另一个 Kk+1Κ_{k+1}Kk+1​ 的一个顶点重合得到的图，那么对于 GGG 的任意两个不相邻顶点 uuu 与 vvv，有：
