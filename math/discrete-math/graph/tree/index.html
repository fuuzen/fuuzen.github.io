# 管梅谷的破圈法

在克鲁斯克尔算法基础上，我国著名数学家管梅谷教授于 1975 年提出了最小生成树的破圈法。

从赋权图 GGG 的任意圈开始，去掉该圈中权值最大的一条边，称为破圈。

不断破圈，直到 GGG 中没有圈为止，最后剩下的 GGG 的子图为 GGG 的最小生成树
