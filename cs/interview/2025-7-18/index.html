# 杂谈

面试官首先看了我的项目经历，聊了其中 3 个项目。

然后面试官看了我的博客，说我博客好像都是 Java、C++ 之类的东西，(内心吐槽：没有 Java 啊？)。接着说我有一些密码学的东西，然后问我 HTTP 和 HTTPS 的区别。

接下来问我这些：

  • 线程和进程有什么区别。
    • 我：线程是轻量级进程，共享堆空间，有独立的栈空间，从主线程 fork 出来开销很小。
    • 面试官：等了好一会，问我还有吗？
    • 我：挠头（还有什么好说的）想不出了。
  • JS 为什么是单线程的，这样有什么好处。
    • 我：JS 设计成事件循环，所以就只能是单线程（实际上好像是反过来），这样有利于浏览器反复刷新渲染的需求（实际上好像和这个无关），然后就是并发的话会有更多开销，object 之类的啊很难管理（想说维护一致性，但没表达出来）。
  • 最新的 ECMAScript 标准是哪一个，讲一下新标准有哪些新特性。
    • 我真不知道啊，新特性一个都说不出来。
    • 面试官：没关系。

接下来做了 4 道 JS 题目。
