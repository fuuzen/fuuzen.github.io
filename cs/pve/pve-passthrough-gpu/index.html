# VM 配置

选中 VM ，在 == 硬件（Hardware）== 部分中，== 机型（Machine）== 应该（尽量）选择 q35 类型，如果不是的话可以在关机状态下修改。

然后还是在硬件（Hardware）部分，点击添加（add），选择 PCI 设备，找到自己的显卡添加设备。
