# 定义

最小生成树 Minimum Spanning Tree 问题

等价问题：寻找一个带权无向图中某一点到另一个点的最短路径 / 权最小的路径

注意：当各边有相同权值时，由于选择的随意性，产生的生成树可能不唯一。

# Kruskal (克鲁斯卡尔) 算法

以点为主导，利用最小堆和并查集

对所有边按权值从小到大排序，逐个添加不形成回路的边，直到包含所有的顶点。

# Prim (普里姆) 算法

  • 以边为主导
  • 贪心思想，不断选最小权值的边
  1. 从某一顶点出发，选择与它关联的具有最小权值的边，将其顶点加入到生成树顶点集合 U1U_1U1​ 中，剩余顶点集合为 U2U_2U2​。
  2. 以后每一步从：
    • 一个顶点在 U1U_1U1​ 中，
    • 另一个顶点在 U2U_2U2​ 中，
      的所有边中权值最小的边，将 U2U_2U2​ 中该顶点加入到 U1U_1U1​ 中
  3. 重复步骤 2 继续 n−1n-1n−1 次直到 U2U_2U2​ 为空

# 两种算法比较

分析普里姆算法，设连通网络有 n 个顶点，则该算法的时间复杂度为O(n2)O(n^2)O(n2)，它适用于边稠密的网络，与边的数目无关。
克鲁斯卡尔算法主要针对边展开，边数少时效率会很高，所以对于边稀疏的图有优势。
