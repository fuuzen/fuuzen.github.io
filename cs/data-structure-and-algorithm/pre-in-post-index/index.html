# 中缀 -> 后缀

  • 从左到右扫描
  • 从右到左打印
  • 遇到操作数：直接打印
  • 遇到操作符：
    1. 优先级小于等于栈顶：栈顶出栈并打印，继续与下一个栈顶操作符比较
    2. 优先级大于栈顶：入栈
    3. 栈顶为括号 (一定是左括号)：入栈
  • 遇到括号：
    1. 左括号：直接入栈
    2. 右括号：不断出栈并打印，直到出栈一个左括号为止

# 中缀 -> 前缀

  • 从右到左扫描
  • 从左到右打印
  • 遇到操作数：直接打印
  • 遇到操作符：
    1. 优先级小于栈顶：栈顶出栈并打印，继续与下一个栈顶操作符比较
    2. 优先级大于等于栈顶：入栈
    3. 栈顶为括号 (一定是右括号)：入栈
  • 遇到括号：
    1. 左括号：不断出栈并打印，直到出栈一个右括号为止
    2. 右括号：直接入栈
